<h1>Sistema de Reconhecimento Facial em Java com OpenCV</h1> <p>Este projeto implementa um sistema completo de reconhecimento facial utilizando Java, OpenCV e JavaCV. Ele permite capturar imagens faciais, treinar modelos de reconhecimento e realizar o reconhecimento em tempo real através da webcam.</p> <h2>Tecnologias Utilizadas</h2> <p>Java</p> <p>OpenCV (via JavaCPP / JavaCV)</p> <p>Haar Cascade para detecção facial</p> <p>Algoritmos de reconhecimento facial: EigenFaces, FisherFaces e LBPH</p> <h2>Estrutura do Projeto</h2> <p>O projeto está organizado em três classes principais, cada uma responsável por uma etapa do processo de reconhecimento facial.</p> <h2>Classe Captura</h2> <p>Responsável por capturar imagens do rosto utilizando a webcam.</p> <p>O sistema solicita o ID da pessoa via terminal e captura automaticamente 25 amostras do rosto detectado.</p> <p>As imagens são convertidas para escala de cinza, redimensionadas para 500x500 pixels e salvas no diretório <strong>src/fotos</strong>.</p> <p>O detector facial utiliza o classificador Haar Cascade.</p> <h2>Classe Treinamento</h2> <p>Responsável por treinar os classificadores faciais a partir das imagens capturadas.</p> <p>O sistema percorre todas as imagens do diretório <strong>src/fotos</strong>, extrai o ID da pessoa pelo nome do arquivo e cria os rótulos correspondentes.</p> <p>Os seguintes algoritmos são treinados:</p> <p>EigenFaces</p> <p>FisherFaces</p> <p>LBPH (Local Binary Patterns Histograms)</p> <p>Após o treinamento, os classificadores são salvos no diretório <strong>src/recursos</strong> em arquivos no formato <strong>.yml</strong>.</p> <h2>Classe Reconhecimento</h2> <p>Responsável pelo reconhecimento facial em tempo real.</p> <p>Utiliza a webcam para capturar imagens, detecta rostos com Haar Cascade e realiza a predição utilizando o classificador LBPH previamente treinado.</p> <p>O nome da pessoa reconhecida é exibido sobre o rosto detectado na tela.</p> <p>Caso o rosto não seja reconhecido, o sistema exibe a mensagem <em>"identidade Desconhecida"</em>.</p> <h2>Fluxo de Funcionamento</h2> <p>1. Executar a classe Captura para coletar imagens faciais.</p> <p>2. Executar a classe Treinamento para gerar os classificadores.</p> <p>3. Executar a classe Reconhecimento para identificar pessoas em tempo real.</p> <h2>Requisitos</h2> <p>Java JDK 8 ou superior</p> <p>Webcam funcional</p> <p>Bibliotecas JavaCV e OpenCV corretamente configuradas</p> <h2>Observações Importantes</h2> <p>O tamanho das imagens deve ser mantido consistente entre captura, treinamento e reconhecimento.</p> <p>O classificador Haar Cascade deve estar corretamente localizado no diretório <strong>src/recursos</strong>.</p> <p>O reconhecimento LBPH utiliza um threshold configurável para controle de precisão.</p><p>Projeto desenvolvido para fins educacionais, baseado em estudos de reconhecimento facial com OpenCV em Java.</p>
